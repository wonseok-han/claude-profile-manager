## v1.0.0 - 2026-04-28

Initial release of claude-profile-manager. 14 files changed, +2147 lines.

### Added

- **Core CLI** — PR #2
  - Profile management: create, clone, rename, remove, status, list
  - Interactive setup wizard and environment doctor
  - Claude Code statusline integration (Dracula theme)
  - Expanded color palette with consistent UI across CLI and installer

- **Credential Refresh** — PR #3
  - `cpm refresh <profile>` for non-interactive token renewal
  - Uses `claude auth status` (~0.2s) without opening interactive session
  - Shell completion support for bash and zsh

- **Uninstall** — PR #4
  - `cpm uninstall` for clean removal of binary, share directory, and rc file blocks
  - Active Claude session detection with warning before proceeding
  - Profile directories preserved by default, separate confirmation required

- **Infrastructure** — PR #1
  - Claude Code commands (/commit, /pr, /changelog)
  - GitHub Actions release workflow (tag → GitHub Release from changelog)
  - Installer supporting macOS, Linux, WSL (curl/git-clone/brew)
